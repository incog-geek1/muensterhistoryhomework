Also read the rules to see what you can ban someone for.

!!warn [user] (reason)

COMING SOON: Warns a user. The user will have to read the warning before being allowed to chat again.

!!ban [user] (reason)

Bans a user. If a reason is not provided, the user will see "No reason given" as the reason. However, we highly suggest you add a reason always to help with appeals.

!!pban [user] (reason)

Bans a user and disables account creation. THIS COMMAND IS IRREVERSIBLE.
If a reason is not provided, the user will see "No reason given" as the reason. However, we highly suggest you add a reason always to help with appeals.

!!unban [user]

Unbans a user.

!!username [user] [new username]

Changes a user's username.

!!chkusr [user]

Gives info about a user. The message can only be seen by you.

!!m [text]

Sends a system message to everyone.
